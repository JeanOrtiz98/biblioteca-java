# 📚 Proyecto Biblioteca en Java

Este proyecto implementa una biblioteca básica utilizando programación orientada a objetos en Java.

## 🚀 Características

- Uso de herencia mediante una clase base `MaterialBiblioteca`
- Implementación de subclases:
  - Libro
  - Revista
  - Tesis
- Aplicación de polimorfismo usando un arreglo de tipo `MaterialBiblioteca`
- Sobrescritura del método `mostrarInfo()`

## 🧠 Conceptos aplicados

- Encapsulamiento (atributos privados con getters y setters)
- Herencia (`extends`)
- Polimorfismo
- Sobrescritura de métodos (`@Override`)

## 🛠️ Tecnologías

- Java 17
- Maven

## ▶️ Ejecución

El programa crea un arreglo de materiales (libro, revista y tesis) y recorre cada uno mostrando su información en consola.
